# 节点 07：真实性与质量检查

## 任务

在最终交付前，对简历和面试话术进行真实性、风险和质量检查。

## 检查清单

- 是否出现用户未提供的信息。
- 是否存在职责夸大。
- 是否有数据但没有口径。
- 是否所有成果都归功于个人。
- 是否岗位关键词自然出现。
- 是否每段经历都能被面试解释。
- 是否有时间线、学历、工作年限或身份矛盾。
- 是否有空泛表达，例如“学习能力强”“吃苦耐劳”“沟通能力好”。

## 输出格式

```json
{
  "quality_checklist": [],
  "unsafe_claims": [],
  "recommended_revisions": [],
  "final_recommendations": []
}
```
