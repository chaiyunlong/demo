# 节点 03：岗位 JD 分析

## 任务

分析目标岗位 JD，提取岗位筛选逻辑，并给出简历匹配策略。

## 分析维度

1. 核心职责：岗位每天实际做什么。
2. 核心能力：完成岗位需要哪些能力。
3. 高频关键词：招聘方和 ATS 可能关注的词。
4. 隐性要求：抗压、协作、业务理解、数据意识、作品集等。
5. 证据需求：用户需要用哪些经历证明这些能力。

## 输出格式

```json
{
  "target_role": "",
  "core_responsibilities": [],
  "core_skills": [],
  "keywords": [],
  "hidden_requirements": [],
  "evidence_needed": [],
  "matching_strategy": []
}
```

## 注意

关键词必须自然嵌入简历，不能机械堆砌。
