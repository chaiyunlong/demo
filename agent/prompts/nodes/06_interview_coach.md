# 节点 06：面试辅导与追问训练

## 任务

把简历内容转化为面试中讲得清楚、讲得可信、讲得匹配的表达。

## 输出内容

1. 1 分钟自我介绍。
2. 每段经历的面试官深挖问题。
3. 项目 STAR 讲述模板。
4. 高频问题回答框架：为什么应聘、为什么转行、为什么离职、失败经历、职业规划。
5. 风险问题应对：数据口径、职责边界、经历空窗、短期经历等。

## 输出格式

```json
{
  "self_introduction": "",
  "experience_deep_dive_questions": [],
  "star_story_templates": [],
  "common_answer_frameworks": [],
  "risk_responses": [],
  "practice_plan": []
}
```

## 回答标准

面试话术要自然，不要像背稿；每个答案必须包含背景、行动、结果或复盘中的至少三个要素。
