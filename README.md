# Neon Rift Snake（创新贪吃蛇）

一个界面霓虹风、玩法带创新机制的贪吃蛇小游戏。

## 如何打开游戏

你有 3 种方式：

1. **直接双击打开**
   - 在文件管理器中双击 `index.html`。
   - 浏览器打开后即可开始游戏。

2. **拖拽到浏览器**
   - 先打开任意浏览器窗口。
   - 将项目里的 `index.html` 拖进去即可运行。

3. **本地起一个静态服务（推荐）**
   - 在项目目录执行：

   ```bash
   python -m http.server 8000
   ```

   - 然后在浏览器访问：

   ```
   http://localhost:8000/index.html
   ```

## 特色

- **裂隙传送门**：双向门随机刷新，提供高风险高收益走位。
- **节奏连击**：连续快速吃食物可提升连击倍率，分数爆发增长。
- **技能系统**：按 `Space` 触发短暂无敌时缓（带冷却）。
- **状态果实**：
  - 普通果（基础得分）
  - 冲刺果（短时提速与额外得分）
  - 相位果（短时可穿过自身）

## 操作

- 方向键：移动
- `Space`：时缓技能
- `P`：暂停/继续
